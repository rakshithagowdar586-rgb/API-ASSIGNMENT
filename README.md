AI API Integration Assignment
Project Overview
This project demonstrates how to integrate and interact with multiple AI APIs using Python.

The following APIs are implemented:
-  Groq API
-  Ollama (Local LLM)
-  Hugging Face API
-  Cohere API

##  Objectives
- Understand API integration with Python
- Work with different AI platforms
- Use environment variables for secure API handling
- Handle errors and API responses effectively

##  Technologies Used
- Python
- REST APIs
- Groq SDK
- Cohere SDK / HTTP Requests
- OpenAI-compatible API (Hugging Face)
- Ollama (Local model)
